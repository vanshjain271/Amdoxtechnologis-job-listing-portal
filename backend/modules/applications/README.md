# Applications Module

This module will handle job applications from Job Seekers.

## Planned Features
- Apply for a job (Job Seeker only)
- View my applications (Job Seeker)
- View applications for a job (Employer)
- Update application status (Employer)
- Withdraw application (Job Seeker)

## Planned Files
- `applicationModel.js` - Mongoose schema for applications
- `applicationController.js` - CRUD logic
- `applicationRoutes.js` - Express routes
