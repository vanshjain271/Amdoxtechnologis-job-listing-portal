# Jobs Module

This module will handle job posting and management features.

## Planned Features
- Create job posting (Employer only)
- List all jobs (Public)
- Get single job details
- Update job posting (Employer only)
- Delete job posting (Employer only)
- Search & filter jobs

## Planned Files
- `jobModel.js` - Mongoose schema for job postings
- `jobController.js` - CRUD logic
- `jobRoutes.js` - Express routes
